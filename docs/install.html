Install¶

Minimum requirement¶

  • SageMath version 9.1 (Release Date: 2020-05-20) or higher.

  • OpenSSL package.

  • 4ti2 package in SageMath.

  • Macaulay2 version 1.14 or higher.

Step-by-Step Installation¶

  1. Install openssl package. You may use package manager like Homebrew.

  2. Install Macaulay2.

  3. Install SageMath.

  4. Install 4ti2 package via

sage -i 4ti2

  1. Install PyOpenSSL package via

sage -i pyopenssl

  1. Install StdPairs package via

sage -pip install stdpairs

Optional Package: PyNormaliz¶

PyNormaliz package is optional and used for sending an affine semigroup to the polyhedron generated by “saturation” of the given affine semigroup. To install this, compile Normaliz by downloading its released version of source code and type

./install_normaliz_with_eantic.sh

on folders containing the source code. Then, you can install pynormaliz via

sage -python -m pip install pynormaliz
