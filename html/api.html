There are three classes AffineMonoid, MonomialIdeal, and ProperPair. A MonomialIdeal object represents an ideal in the affine monoid, and a proper pair represent a pair of a monomial (element) in the affine monoid and a face on the affine monoid, which is not contained in some fixed ideal. See Yu, 2020 or Matusevich and Yu, 2020 for details in mathematical context.
